Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML it will show the Whole HTML code with content, innerText : it will show only the visible text inside the element , if any text is display none it wont show that, textContent: it will show the text also the hidden or display none text also will be shown.
